PHASE 1: ARCHITECTURE VERIFICATION

What to ask:

"Show me the actual code for Stage 4 (OBMP Algorithm). I want to see:

The deterministic conflict resolution logic
How you compute pull-before events
How you handle medication timing (Metformin example)
The modifierSection JSON structure that gets injected into Gemini

Format: GitHub repo link OR screenshare walkthrough (30 min)"

What you're checking:
