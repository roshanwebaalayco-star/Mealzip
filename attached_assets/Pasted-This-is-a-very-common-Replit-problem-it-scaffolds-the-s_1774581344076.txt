## Fix 2 — PostgreSQL connection (1 hour)

The schema exists but the database is not connected. In Replit Secrets add:

```
DATABASE_URL = postgresql://user:password@host:5432/dbname
```

Replit provides a free PostgreSQL database. Go to **Tools → Database** in Replit and it will generate this URL automatically. Then tell Replit:

*"Connect the DATABASE_URL secret to the Drizzle ORM configuration. Run `drizzle-kit push` to create all tables from the schema. Confirm the connection works by adding a health check that queries `SELECT 1` from the DB on server start."*
