# ParivarSehat AI — Conversational Voice Assistant
## Development Prompt

---

## WHAT TO BUILD

Replace the current "Speak Profile" one-shot button with a **fully conversational voice assistant** that behaves exactly like Siri or Google Assistant — asking one question at a time, listening for the answer, confirming what it heard, then moving to the next question. The form fills itself silently in the background as the conversation progresses.

---

### Core Hook: useVoiceAssistant

```javascript
const useVoiceAssistant = () => {
  const [state, setState] = useState(CONVERSATION_STATES.GREETING);
  const [transcript, setTranscript] = useState('');
  const [isListening, setIsListening] = useState(false);
  const [isSpeaking, setIsSpeaking] = useState(false);
  const [formData, setFormData] = useState(initialFormData);
  const [currentMember, setCurrentMember] = useState({});
  const [conversationHistory, setConversationHistory] = useState([]);

  // Main conversation engine
  const processUserResponse = async (userText) => {
    // Add to history
    addToHistory('user', userText);
    
    // Send to Gemini with current state context
    const parsed = await parseWithGemini(userText, state, formData);
    
    // Update form data silently
    updateFormData(parsed);
    
    // Determine next state
    const nextState = getNextState(state, parsed);
    setState(nextState);
    
    // Generate and speak assistant response
    const response = generateResponse(nextState, parsed, formData);
    await speak(response);
    addToHistory('assistant', response);
    
    // If complete, submit form
    if (nextState === CONVERSATION_STATES.COMPLETE) {
      submitForm(formData);
    }
  };
  
  return { state, isListening, isSpeaking, 
           conversationHistory, formData, startListening };
};
```

### Sarvam STT Integration (Speech → Text)

```javascript
const startListening = async () => {
  setIsListening(true);
  
  // Start recording audio
  const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
  const recorder = new MediaRecorder(stream);
  const chunks = [];
  
  recorder.ondataavailable = (e) => chunks.push(e.data);
  recorder.onstop = async () => {
    const audioBlob = new Blob(chunks, { type: 'audio/wav' });
    const transcript = await transcribeWithSarvam(audioBlob);
    setTranscript(transcript);
    await processUserResponse(transcript);
  };
  
  // Auto-stop after silence (2.5 seconds)
  recorder.start();
  setTimeout(() => {
    recorder.stop();
    stream.getTracks().forEach(t => t.stop());
    setIsListening(false);
  }, 6000); // max 6 seconds per response
};

const transcribeWithSarvam = async (audioBlob) => {
  const formData = new FormData();
  formData.append('file', audioBlob, 'audio.wav');
  formData.append('model', 'saarika:v2.5');
  formData.append('language_code', 'hi-IN'); // auto-detect
  
  const response = await fetch('https://api.sarvam.ai/speech-to-text', {
    method: 'POST',
    headers: { 'api-subscription-key': process.env.SARVAM_API_KEY },
    body: formData
  });
  
  const data = await response.json();
  return data.transcript;
};
```

### Gemini Parsing — Maps Speech to Form Fields

```javascript
const GEMINI_PARSE_PROMPT = `
You are a form-filling assistant for an Indian family health app.
Current conversation state: {STATE}
Current partial form data: {FORM_DATA}

The user just said: "{USER_TEXT}"

Extract ONLY what the user said in this turn. Return strict JSON:
{
  "family_name": null or "string",
  "state": null or "Jharkhand|Bihar|UP|...",
  "monthly_budget": null or number,
  "language": null or "Hindi|English|Bengali|...",
  "dietary_type": null or "vegetarian|non-vegetarian|vegan|jain|eggetarian",
  "member": {
    "name": null or "string",
    "role": null or "self|spouse|father|mother|son|daughter|grandparent",
    "age": null or number,
    "gender": null or "male|female",
    "conditions": null or ["diabetes","hypertension","anemia","obesity","thyroid","pcod","none"],
    "health_goal": null or "weight_loss|diabetes_management|general_wellness|child_growth|muscle_gain"
  },
  "add_more_members": null or true or false,
  "ready_to_generate": null or true or false
}

MAPPING RULES (Hindi → English):
- paanch hazaar / 5000 → 5000
- madhumeh / sugar ki bimari → diabetes
- BP / blood pressure / uchch raktachap → hypertension
- khoon ki kami / anemia / raktalpata → anemia
- shakahari → vegetarian
- maansahari → non-vegetarian
- pitaji / papa / dad → father
- maa / mummy / mom → mother  
- beta / ladka / putra → son
- beti / ladki / putri → daughter
- dada / dadi / nana / nani → grandparent
- pati / husband → spouse
- patni / wife → spouse
- haan / ji haan / yes → true
- nahi / no / bas → false
- aur nahi / sirf itne / ho gaya → ready_to_generate: true
`;

const parseWithGemini = async (userText, currentState, formData) => {
  const prompt = GEMINI_PARSE_PROMPT
    .replace('{STATE}', currentState)
    .replace('{FORM_DATA}', JSON.stringify(formData))
    .replace('{USER_TEXT}', userText);
    
  const response = await callGemini(prompt);
  return JSON.parse(response);
};
```

### Text-to-Speech (Browser Native — Free)

```javascript
const speak = (text) => {
  return new Promise((resolve) => {
    window.speechSynthesis.cancel(); // stop any current speech
    
    const utterance = new SpeechSynthesisUtterance(text);
    utterance.lang = formData.language === 'Hindi' ? 'hi-IN' : 'en-IN';
    utterance.rate = 0.9;  // slightly slower = clearer
    utterance.pitch = 1.1; // friendly tone
    
    utterance.onend = () => {
      setIsSpeaking(false);
      // Auto-start listening after assistant finishes speaking
      setTimeout(() => startListening(), 300);
      resolve();
    };
    
    setIsSpeaking(true);
    window.speechSynthesis.speak(utterance);
  });
};
```

  const fallbacks = {
    ask_family_name: "Koi baat nahi — sirf apna family naam bolein, jaise 'Sharma' ya 'Gupta'",
    ask_budget: "Mahine mein khane par kitne rupaye kharch karte hain? Jaise '3000' ya 'paanch hazaar'",

  };
  speak(fallbacks[state] || "Kya aap dobara bol sakte hain?");
};

  setVoiceMode(false);
  // Show standard form with data already filled
};

## SUCCESS CRITERIA

The assistant must:
- Ask questions naturally, one at a time
- Understand Hindi + English + mixed (Hinglish)
- Confirm each answer before moving on
- Fill the form silently in background
- Handle "wait" / "ruko" / corrections gracefully
- Complete full family profile in under 3 minutes
- Auto-navigate to meal plan when complete
- Always offer "type instead" escape hatch

---

## WHAT NOT TO BUILD
- Do NOT require internet for TTS (use browser Web Speech API)
- Do NOT make user repeat themselves (confirm before moving on)
- Do NOT ask more than 2 questions at once
- Do NOT show raw JSON or error logs to user ever
